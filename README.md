# 🎬 Real-Time Movie Ticket Booking System

A full-stack movie ticket booking web application developed using PHP, MySQL, HTML, CSS, and JavaScript.

## 🚀 Features

- User Signup & Login
- Session Management
- Dynamic Movie Listings
- Balcony & Nela Seat Booking
- Real-Time Seat Availability
- Duplicate Seat Booking Prevention
- QR Payment Demo Page
- Booking Confirmation Page
- Responsive UI Design

## 🛠 Technologies Used

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP

## 🎟 Seat Categories

- Balcony Seats
- Nela (Down) Seats

## 💳 Payment

Demo QR payment system included.
